# Web-Scraping-Data-Pipeline
# 🕸️ 방치 웹사이트 불법 게시글 유통 구조 추적 파이프라인

## 📌 프로젝트 개요
관리가 끊긴 대학 게시판 등 방치된 웹사이트가 마약, 위조 신분증 등 불법 광고의 인프라로 악용되는 실태를 고발하기 위한 데이터 탐사 프로젝트입니다. 분석의 기반이 되는 41만여 건의 방대한 비정형 웹 텍스트 데이터를 안정적으로 확보하기 위해 크롤링 및 전처리 파이프라인을 직접 구축했습니다.
* **수상 내역:** SBS 데이터사이언스와저널리즘아카데미 4기 최우수상 수상작 (팀 프로젝트)

## 🛠 사용 기술 (Tech Stack)
* **Language:** Python
* **Data Collection:** BeautifulSoup, Selenium, Requests
* **Data Processing:** Pandas, Regex (정규표현식)

## 💡 코드의 핵심 역할 및 수행 업무 (수집 및 전처리)
1. **대규모 비정형 데이터 수집 자동화**
   * 활성화된 65개 방치 사이트의 구조를 분석하고, 41만 6천여 건의 게시글 원문, 제목, 작성일, 작성자 데이터를 긁어오는 웹 크롤러 설계.
   * 사이트별로 다른 HTML 구조와 접근 제어를 우회하고 안정적으로 데이터를 적재할 수 있도록 파이프라인 최적화.
2. **데이터 전처리 및 텍스트 정제**
   * 수집된 텍스트 내의 특수문자, HTML 태그, 불필요한 공백 등을 정규표현식을 활용해 1차 정제.
   * 이후 팀원들이 형태소 분석(Kiwi) 및 범죄 유형 자동 라벨링(Naive Bayes) 모델을 원활하게 적용할 수 있도록 깨끗한 데이터셋(Clean Dataset)을 제공하는 핵심 엔지니어링 역할 수행.

## 📊 시사점
파이프라인을 통해 빠르고 정확하게 정제된 원천 데이터를 바탕으로, 탐사기획팀이 '범죄 유형-텔레그램 아이디' 간의 네트워크를 생키 다이어그램으로 도출할 수 있었으며, 이는 실제 마약 거래 위장 잠입 취재로 이어지는 결정적 단서가 되었습니다.
